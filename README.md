# 宠物塔罗占卜 Demo

本项目是用 React + Vite 实现的简单交互式宠物塔罗占卜 Demo。

运行：

1. 安装依赖：

```bash
npm install
```

2. 在根目录创建 `.env` 文件，添加：

```
VITE_GEMINI_API_KEY=your_api_key_here
```

3. 本地开发：

```bash
npm run dev
```

说明：
- Gemini API 调用使用 `VITE_GEMINI_API_KEY` 环境变量，免费额度可参考 [Google AI Studio](https://ai.google.dev/)。
- Demo 仅包含 22 张大阿尔卡那样例数据，完整版需补充56张小阿尔卡那。
